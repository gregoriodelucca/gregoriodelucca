### 🧩 Sobre mim
Sou **Desenvolvedor .NET** com experiência no desenvolvimento de **aplicações web e sistemas ERP**, atuando na criação e evolução de soluções para gestão empresarial.

No **back-end**, trabalho com **.NET e C#**, desenvolvendo **APIs, regras de negócio e integrações entre sistemas**, além de manipulação de dados e arquitetura de aplicações.  
No **front-end**, utilizo **React** para criação de interfaces modernas e responsivas integradas às aplicações.

Tenho experiência no desenvolvimento de **sistemas corporativos**, especialmente **ERPs para varejo e soluções para pet shops**, contribuindo na implementação de funcionalidades e melhorias em aplicações voltadas para operações de negócio.

🧠 **Pessoa com Deficiência (PcD)** • Perfil analítico, organizado e em constante evolução técnica.
